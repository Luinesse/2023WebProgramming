## 웹페이지의 구성

> 문제에서 요구한 조건은 다음과 같습니다.

+ 라디오 버튼 클릭시 이미지 교체

## 자바스크립트 작성

-----------------------------

> 각 버튼의 onchange 속성을 imgChange 함수로 설정합니다. imgChange 함수는 for문으로 버튼을 검사하며 체크가된 버튼의 value값을 이미지의 src 속성으로 설정합니다.
