## 웹페이지의 구성

> 문제에서 요구한 웹페이지는 다음 조건을 만족해야합니다.

+ 수식을 입력받아 eval() 함수를 이용하여 계산 결과를 출력

## eval() 함수로 계산하여 출력

-----------------------------

> prompt() 함수를 통해 수식을 입력받고, 입력받은 수식을 eval() 함수에 파라미터로 전달하여 값을 출력합니다.
