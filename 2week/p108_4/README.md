### HTML 파일의 구성

-----------------------------

> 해당 문제에서는 다음과 같은 조건을 충족해야합니다.

-----------------------------

> 테이블의 제목은 table 태그 안에서 caption 태그를 사용하여 중심에 위치하게끔 구현했습니다.

-----------------------------

> 각 열의 제목은 thead 태그를 활용하여 열을 대표하는 행으로 보이게 구현했습니다.
